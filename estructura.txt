Flujo de Ejecución del Código
El flujo de ejecución del código sería algo como esto:

El archivo main.py se ejecuta.
main.py carga el menú principal de menu_principal.py.
Cuando el usuario hace clic en "Ir a Información", se llama a menu_info.py.
Cuando el usuario hace clic en "Ejecutar Entorno Gym", se llama a ejecutar_gym.py para iniciar el entorno de OpenAI Gym.
El código se organiza de manera modular, lo que facilita su mantenimiento y escalabilidad.
